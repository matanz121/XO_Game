XO Game:

1) Netanel Abutbul & Eliyahu Vaknin - 208207027 & 208209767

2) HTML, CSS and JavaScript seperated.

3) The game end when all the table if fully colored and give the player alert with the score.

4) When the game end you have the option to restart the game and choose again the cols & rows and start to play.

5) Every 4 cells with the same color will give you 1 point.

6) The game is available to download on - https://github.com/matanz121/XO_Game

Enjoy & Good luck.
